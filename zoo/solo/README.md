### Display a file
```
COPY (SOLOCOPY, CONSOLE)
CONSOLE:
MOVE(1)
WRITE(AUTOLOAD)
BACKUP(WRITE)
MOVE(2)
BACKUP(CHECK)
MOVE(1)
```

### Copy a file
```
COPY(SOLOCOPY, SOLOCOPY2)
```
